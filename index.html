About Me

My name is Kourken Kasbarian, I'm an undergraduate IT student and a Front-End Web Developer from Armenia. I love to create and build websites which are mobile-first ready, fully responsive, minimal, SEO friendly and at the same time attractive and user-friendly for individuals, startups, businesses etc. I built my portfolio site from the grounds up by using all the skills that I know. I also have some fairly basic knowledge of Web/Graphic Design. I'm always motivated and have the desire to learn more.

While I'm not debugging some code or trying to position elements right, I love to hang out with my friends, play video games, listen to music, photograph-ing, eating pizzas etc.
